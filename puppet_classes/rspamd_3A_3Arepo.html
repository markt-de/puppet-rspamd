Class: rspamd::repo

This class adds a package repo to your system's package manager.

So far supported are:

  • Debian/Ubuntu (adds repo::apt_stable)

Examples:

include rspamd::repo

Parameters:

  • baseurl (Any) (defaults to: undef) —

    use a different repo url instead of rspamd.com upstream repo

class rspamd::repo (
  $baseurl = undef,
) {
  case $::osfamily {
    'Debian': {
      class { 'rspamd::repo::apt_stable': }
    }

    default: {
      fail("Unsupported managed repository for osfamily: ${::osfamily}, operatingsystem: ${::operatingsystem}, module ${module_name} currently only supports managing repos for osfamily Debian")
    }
  }
}
